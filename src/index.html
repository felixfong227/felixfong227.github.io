A Story About Myself

I'm a self-taught software engineer 👨‍💻

Most of my learning experience came from YouTube tutorials and continues to be so.

From time to time, I just decide on the whim to build some truly ridiculous and silly projects 😝

From all these personal projects, I am hoping to familiarize myself with new technologies along the way.

I have been experimenting with web development from around 2015, starting from making basic HTML pages all the way to programming some of the most well-known server-side programming languages like NodeJS, PHP and Go 👈🏻

Lately, I also got involved with DevOps tooling, like Docker 🐬 and how to set up an Nginx server over a TLS 🔐 connection. Funny thing is, I do enjoy doing those! 🤭
